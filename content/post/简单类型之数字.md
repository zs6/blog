## 数字

## 基本数字类型

```java
List<Number> list = new ArrayList<Number>();  
list.add((byte)42); //byte  
list.add(512); //int  
list.add((short)1024); //short  
list.add((long)777); //long  
list.add(new BigInteger("9876543210"));  
  
for (Number n : list) {  
System.out.println(n);  
}
```

## 小数类型

### 大范围小数

```python
from decimal import Decimal  
  
a = Decimal(270000000000)  
# a is 270000000000  
  
a += 2  
# a is 270000000002  
  
a /= 1000  
# a is 270000000.002
```

```java
BigDecimal a = new BigDecimal("27e10");  
//a is 2.7E+11  
  
BigDecimal b = BigDecimal.valueOf(2);  
BigDecimal c = BigDecimal.valueOf(1000);  
  
a = a.add(b);  
//a is 270000000002  
  
a = a.divide(c);  
//a is 270000000.002
```
